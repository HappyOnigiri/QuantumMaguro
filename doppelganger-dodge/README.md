# Doppelganger Dodge

ブラウザベースの「避けゲー（ドッジゲーム）」です。

## 概要

プレイヤーは青い円をマウスで操作し、迫り来る赤い敵を避けながら、黄色のスコア（コイン）を集めます。

## 狂った独自ルール：【ドッペルゲンガーの呪い】

このゲームの最大の特徴は、**スコアを取るたびに自分の動きを模倣する分身（ドッペルゲンガー）が出現する**ことです。
出現した分身の「どれか一つ」でも敵に触れると、その瞬間にゲームオーバーになります。

### 分身のパターン
分身はスコアを取るたびに以下の順番で追加されていきます。
1. **左右反転**: マウスの左右の動きと逆方向に動きます。
2. **上下反転**: マウスの上下の動きと逆方向に動きます。
3. **完全反転**: 点対称に動きます。
4. **遅延（0.3秒）**: 少し前のあなたの動きをなぞります。
5. **遅延（0.6秒）**: さらに前のあなたの動きをなぞります。
6. **X軸・Y軸入れ替え**: マウスを横に動かすと縦に、縦に動かすと横に動きます。
