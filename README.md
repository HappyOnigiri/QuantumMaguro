## ローカル環境での起動手順

ローカル環境でプロジェクトを起動する手順：

```bash
# リポジトリをクローン
git clone git@github.com:HappyOnigiri/game-portal

# プロジェクトディレクトリへ移動
cd game-portal

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```
