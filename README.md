## ローカル環境での起動手順

ローカル環境でプロジェクトを起動する手順：

```bash
# リポジトリをクローン
git clone git@github.com:HappyOnigiri/QuantumMaguro

# プロジェクトディレクトリへ移動
cd QuantumMaguro

# 依存関係をインストール
pnpm install

# 開発サーバーを起動
pnpm run dev
```
